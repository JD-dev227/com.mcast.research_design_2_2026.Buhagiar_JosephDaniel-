This repository contains research papers, LaTeX source files, and supporting material for my final-year dissertation and research modules.

BSc (Hons) Software Development

## Research Topic
Development of a wearable in-shoe system for gait anomaly detection using FSR sensors and an IMU (MPU6050) integrated with an ESP32 microcontroller.
